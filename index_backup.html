Equipo de protección
LA SEGURIDAD ES PARTE IMPORTANTE DE LAS OPERACIONES. LA SEGURIDAD NO ES UNA OPCION. Nuestro mayor compromiso es ofrecer un equipo de protección personal que asegure una protección EFECTIVA Y DE CALIDAD al personal que lo utilice.

Contamos con una diversidad de equipo de protección personal industrial para la diferentes riesgos y áreas tales como:
●Equipo para espacios confinados.
●Botas para materiales peligrosos
●Protección para la cabeza
●Protección auditiva
●Vestimenta de alta visibilidad
●Protección contra el frío
●Equipo para medir explosividad
●Trajes A, B para materiales peligrosos
●Protección para trabajos en altura
●Protección para manos y antebrazos
